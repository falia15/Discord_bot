# Bot discord

A simple discord bot, bluid with [Discord.js API](https://discord.js.org).

## Initialisation

```
npm install
```

Edit _config.exemple.json_ and _src/data/joke.exemple.js with your own value and remove the _.exemple_ from the files names.

Run the bot with

```
node src/app.js
```

## Requirements

Nodejs 11.3.2
